# @ydant/canvas

## 0.3.0

### New Package

Canvas2D rendering backend for Ydant.

- `createCanvasBackend(canvas)`: Canvas2D backend with `tree` capability
- `createCanvasPlugin()`: Processes canvas element and paint spells
- VShape tree structure for canvas scene graph
- `paint()` spell for immediate-mode drawing
- Marker-based reactive container support
