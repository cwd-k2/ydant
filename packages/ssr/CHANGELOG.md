# @ydant/ssr

## 0.3.0

### New Package

Server-side rendering and hydration for Ydant.

- `createSSRBackend()`: SSR backend with `tree`, `decorate`, `resolve` capabilities
- `renderToString()`: Renders app to HTML string
- `hydrate()`: Client-side hydration with DOM node reuse
- VNode-based serialization
